Google Cloud Development Best Practices Curated Links Collection
These best practices are programmatically gathered directly from Google. A machine gets the list of Google Cloud products from cloud.google.com/products and then iteratively searches site:cloud.google.com for each products best practices. Only the top ten results are displayed.
Google Cloud Interconnect Locator
A server periodically calls the listAsync method on Google's Node.js compute engine's InterconnectLocationsClient, geocodes the addresses, then stores the list in Firebase's Realtime Database. As the maps bounds_change event initializes and fires, a geofire geoquery is performed. Google Cloud Interconnect locations shown below. Click the location in the map for more detail.
